이번에는 Pykrx 와 FinanceDataReader를 사용한 금융 데이터 수집을 해보려 한다. GCP를 하면서 ETL에 좋은 예제가 무엇이 있을까 생각하던 도중 금융 데이터를 사용하는 것이 생각이 났다. Pykrx나 FinanceDataReader는 종종 찾아서 쓰던 라이브러리 였으나, 이번에 정리도 하면서 블로그를 남기려고 한다. 둘중 하나만 사용해볼까 했는데 국내, 국외 주식 및 여러 세부정보들이 각각 다르게 제공을 하기 때문에 두개를 써야 하는 상황이 생겼다. 하나씩 정리해보자.

Pykrx

  • 공식문서: Pykrx

사용할 데이터

  • 사용 예정 데이터
    • 국내종목시세
    • 종목별 DIV/BPS/PER/EPS 조회
    • 일자별 시가총액
    • 일자별 거래실적 추이(거래대금)
    • 일자별 거래실적 추이(거래량)
  • 추후 사용
    • 투자자별 거래실적 추이 (거래대금)
    • 투자자별 거래실적 추이 (거래량)

FinanceDataReader

  • 공식문서: FinanceDataReader

사용할 데이터

  • 사용 예정 데이터
    • 국내,해외 종목 리스트
    • 해외주식 가격 데이터
    • 환율 데이터
    • 비트코인 가격

파이썬을 사용한 데이터 수집 가이드

설치

$ pip install pykrx  # pykrx 설치
$ pip install -U finance-datareader # finance-datareader 설치
